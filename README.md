# HimFit

12-week half marathon + strength training program. Mobile-first PWA-ready web app.

## Run locally

Open `index.html` in a browser or serve with any static server:

```bash
# Python
python3 -m http.server 8080

# Node (npx)
npx serve .

# Then visit http://localhost:8080
```

## Features

- **Overview** — Program phases, weekly split, protocols (sauna, creatine, run order, sugar)
- **Today** — Day-by-day workout with exercises, reps, video demos
- **Plan** — 12-week grid with progress dots
- **Runs** — Strava embed, manual run logging (distance, time, HR, RPE)
- **Fuel** — Nutrition plans by session type (lift, run, KB, long run)

Data is stored in `localStorage`. Add to home screen for app-like experience.
